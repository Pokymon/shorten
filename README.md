# Shorten

Shorten is a web application crafted to simplify the process of URL shortening. With a clean, intuitive interface, Shorten makes it easy to convert long URLs into short, manageable links. This is particularly useful for sharing links in a more user-friendly manner across different platforms.

## Installation

This project is built with Django and uses Pipenv for managing dependencies. Follow these steps to get it running on your local machine:

1. Clone the repository to your local machine using `git clone https://github.com/Pokymon/shorten.git`.

2. Navigate to the project directory with `cd shorten`.

3. Run database migrations with `python manage.py migrate`.

4. Create an admin user for the Django admin interface using `python manage.py createsuperuser`.

5. Start the local development server with `python manage.py runserver`.

Now you can access the Shorten web application locally.
